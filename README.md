# BLOGGER

**Functionality**
  * Authentication
  * CRUD Articles
  * CRUD Comments
  * CRUD Tags from Articles
  * CRUD Authors
  * Upload and store images
  * Count views from articles

**Technologies**
  * Active Storage
  * AWS S3
  * Sorcery
  * Postgresql
  * Heroku

**Deploy: https://niconibloggerapp.herokuapp.com**
